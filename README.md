# AAC Noise Modeling

## Description

This repository contains tools and scripts for audio analysis, specifically focusing on sound visualization and level measurements. The project aims to extract and visualize various metrics from audio files, ensuring accurate representation and calibration.

## Table of Contents

- [Features](#features)
- [Contributing](#contributing)
- [License](#license)

## Features

- **SPL/leq_level**: Tools and scripts related to Sound Pressure Level measurements and analysis.
- **sound_visualization**: Scripts and utilities to visualize various aspects of sound, such as spectrograms.
